Big Blue Queue

BBQ it's an ambicious project that was composed by three things

1. An Eclipse plugin for Lotus Notes 8.5 or higher (JAVA SWT)

2. A Java EE Web application ( a kind of request hub, built with Emergys Web-frame 4)

3. An API (built with Java EE components, JAX-WS or SOAP web services and JAX-RS for REST Services) ready to be consumed by other services or software

BBQ Solution is a requests manager for anyapplication. BBQ can apply any kind of actions to any kind of requests (worflow actions) of any application well configured, subscribed and WS clients implemented. In this Project the assignments were: software analyst for the entire solution (infrasturcute and business logic) and developer (build the entire solution, web apps, web and rest services).

Project Challenges

Desing the REST and Web Services API to keep the plugin and the subscribed applications to BBQ well synchronized. Desing & crearte Web services (WSDL) and REST Services (JAX-RS). Eclipse java plugin built in SWT and lotus notes expeditor tools.

Design patterns implementations (Singleton, Factory method)
